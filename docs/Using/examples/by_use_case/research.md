!!! tip "[Open Deep Research](https://github.com/langchain-ai/open_deep_research)"
    A web research assistant that generates comprehensive reports on any topic following a workflow similar to OpenAI and Gemini Deep Research. Key features:
    
    - Customizable models, prompts, report structure, search API, and research depth
    - Allows providing an outline with a desired report structure
    - Configurable planner model (e.g., DeepSeek, OpenAI reasoning model)
    - User feedback on report sections with iteration until approval
    - Configurable search API (Tavily, Perplexity, Exa, ArXiv, PubMed, Linkup)
    - Adjustable search depth for each section
    - Customizable writer model (e.g., Anthropic, OpenAI, Groq)
    - Built on LangGraph with human-in-the-loop approval workflow

## Commercial

!!! note "[Perplexity Deep Research](https://www.perplexity.ai/)"
    Launched in February 2025, Perplexity Deep Research is a powerful research tool that:
    
    - Uses a proprietary framework called "test time compute (TTC)" to explore complex topics
    - Performs dozens of searches and evaluates hundreds of sources automatically
    - Synthesizes findings through probabilistic reasoning models
    - Offers 5 free daily queries for non-subscribers
    - Provides 500 daily queries for Pro subscribers ($20/month)
    - Generates reports in 2-4 minutes with citations and source verification

!!! note "[OpenAI Deep Research](https://openai.com/)"
    Available to ChatGPT Pro users, OpenAI Deep Research is an AI agent that:
    
    - Uses a version of the OpenAI o3 model optimized for web browsing and data analysis
    - Completes research tasks in 5-30 minutes that would typically take hours
    - Employs reasoning to search, interpret, and analyze diverse online sources
    - Provides thorough documentation with citations and summaries
    - Excels at uncovering niche information requiring extensive browsing
    - Targets professionals in finance, science, policy, and engineering
